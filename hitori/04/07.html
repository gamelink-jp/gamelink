2004-07-25 日曜日 編集

■ CPU 載せ替え

7月12日に書いたAthlonXP2400+が届いたので、早速載せ替えを行った。

正常に動作をし、しばらく動かしてからCPU の温度を測ったところ、60℃前半だった。熱暴走が起こっていたときにはケースを開けた状態で60℃後半あったので、まあこれくらいで良いんだろう。

ただ、ちょっと高めなので、以前使っていたケース用のファンをまた付けようかと思っている。

　

この載せ替えによりそれまで使っていた Athlon1800+ があいたので、実家で使っているマシンの Duron1.2GHz と載せ替えることにした。

載せ替えを行ったところ、全く起動しない・・・。

FSB を 133MHz にすると起動しない。

実家で使っているマザーボードは ECS の K7S5A なのだが、このマザーボードは以前、この Athlon1800+ を使っていて上手く動作しなくて買い換えた経緯がある。

その時の症状は、コールドブートに失敗、リブートすると起動すると言ったものだった。今回は全く起動しないのだが、根本原因は同じなのだろう。

その頃とはメモリが SD-RAM から DDR-SD-RAM になったり、ビデオカードが違ったり、結構構成が違うのでやっと原因がわかった。

Duron1.2GHz は FSB が 100MHz で正常動作、Athlon1800+ は FSB 133MHz で正常に動作せず、本来の設定よりも落として 100MHz にして見ると正常に動作した。

BIOS の設定もいくらか試してみたのだが、このマザーボード FSB 133MHz で正常に動作しない不良品なのだろう。

今更、ECS 等に問い合わせる気も無いので、このまま FSB 100MHz で使おうかと考え中。

Duron1.2GHz から Athlon1800+ は、本来 1.52GHz が FSB 100MHz で 1.14GHz ほど、周波数は以前よりわずかに低くなる。また Duron に付け替える気は無いし、 Duron と Athlon の差もあるからまあ、このままで良いか。

　

もう Athlon1800+ は結構古い製品になるので、インターネットオークションなどで安いマザーボードを買おうかと思って、早速検索してみた。

そこで、見ていて気になったのがシリアルポートの数。

1つだけ搭載のボードが多い。

最近はシリアルポートを使うことは少なくなり、1つだけ搭載なのはもはや当たり前、1つも搭載していないボードもある。

なぜシリアルポートの数が気になるかというと、現在の K7S5A はシリアルポートを2つ搭載していて、電話でチケットを予約するときにシリアルポート2つにモデムをそれぞれ付けて2回線使って電話をかけることをすることがあるから。

これがシリアルポートが1つだと電話回線は2つ使えるのにモデムを1つしか繋げられなくなり1回線しか使えなくなる。

シリアル-USB 変換ケーブルを使えば良さそうだが、やはり変換ケーブルを買うと数千円かかりそう。

マザーボードと合わせて1万円近くかかるかも。

Athlon1800+ FSB100MHz 動作でも不便は感じていないので、このまま使うか。

■ 東京ディズニーランド

7月17日にバスツアーを使って東京ディズニーランドに行ってきた。東京ディズニーランドは去年10月に行って以来。

7月は浴衣キャンペーンなるものをやっていたので、浴衣を着て行ってきた。

浴衣は今回初めて買って初めて着た。初めて浴衣を着るのがディズニーランドとは。浴衣を着ている人は思ったよりも少なくてちらほらと見かける程度だった。

　

開園から閉園までたっぷり遊べると言うツアーだったが到着したのは開園時間の8:00、入園したのは8:20頃だった。

　

現在行われている、スペシャルイベント「バズ・ライトイヤー夏の大作戦」を見るためのガイドツアーを申し込もうと思い、受付まで行ってみたが既に売り切れだった。すぐに売り切れると言う話しは聞いていたので、あきらめて、4月にできた新しいアトラクションのバズ・ライトイヤーのアストロブラスターのファストパスを取ることに。並んだ末、14:00頃のファストパスを獲得。

　

最初に乗ったアトラクションは、イッツ・ア・スモールワールド、「世界はひとつ」は良い歌ですね。ディズニーランドにきたら唯一必ず乗りたい一品です。

　

ウエスタンランドで遅めの朝食。カモにパンをあげながらマッタリ過ごす。マッタリ過ごす。。マッタリ過ごす。。。

　

2つめに乗ったアトラクションは、カリブの海賊。こんな暗かったっけ?前に乗ったときよりも暗く感じた。今回くらい暗い感じの方が良いのだが、前もこのくらいだったかな・・・。

　

次は、ショー観覧。バズ・ライトイヤー夏の大作戦。

何が特徴かと言うと、ゲストに向かって水をぶっ掛けるのが特徴。なんと言うか、アメリカ的。一昔前の日本ではできなかっただろうなあと思いながら、喜びながら水をぶっ掛けられた。。

　

3つめのアトラクションはバズ・ライトイヤーのアストロブラスター。他の遊園地でよく見かけるシューティングタイプのアトラクション。ファストパスなら良いけど、スタンバイで2時間とか待ってまで乗る気にはならないかな・・(^_^;)

　

次に、ディズニー・ドリームス・オン・パレード。これは前からやっているパレード。一番前で見た。プレミアムツアーとか以外で、パレードを一番前で見たのははじめてかも。

パレードが停まったときに、マッドハッターとアリスが目の前だったので、写真を沢山撮ってみた。

　

4つめのアトラクション、ビッグサンダー・マウンテン。

乗った場所は真ん中と言う非常に微妙な場所。

ウオッホー♪と叫びながら乗ってきた。

一応絶叫マシンだけど、子供でも乗れる程度のもので、むしろ子供向けのアトラクションなのかも。

「絶叫」を求めるならスプラッシュマウンテンの方がよさげ。

　

再び、バズ・ライトイヤー夏の大作戦を観覧。抽選で入ることができる鑑賞エリアが当たっていたのでそこから鑑賞。「ずぶ濡れエリア」と言われていたが、風向きの影響かほとんど濡れず。なんだか不満だぞ(-_-;ずぶ濡れになりたかったぞ(-_-;

次のショーは、エレクトリカルパレード昔からあるパレード。なのによくもまあ、こんなに人が集まるなあ・・・。人のこと言えないが・・・。

なんだか歳を取ったなあと実感。

子供の頃はいかに沢山のアトラクションに乗るかと言うのが重要だったが、今はマッタリ過ごすのが良い。

■ 父のパソコン

父はこれまでISDNの iプランサービスを使ってインターネットに接続していた。

iプランとは一定時間までは固定料金で、それ以上は従量制になる半固定のインターネット接続サービス。

今回、定年退職したことによりパソコンをいじる時間が多くなったため、フレッツADSLの40Mbpsサービスに変更をした。

もう何年も私は父のパソコン関連に首を突っ込むことは無く、父が一人でやっていたのだが、フレッツADSL にしてからインターネットに接続できなくなったと相談を受けた。

原因は、LANカードが認識できていなかったこと。

父は私に相談する前に、インターネットへの接続設定を色々試していたようだが、そもそも ethernet がリンクアップしていないのだから繋がるわけも無い。

父のパソコンは5年ほど前に私が使っていたパソコンをそのまま引き継いで使っている。

その時点では OS は確か Windows98 だったと思う。その後、父が自分で Windows2000 にアップデート、その時点で既に LANカードが認識できていなかったと思われる。父はこれまで、RS-232C によるダイヤルアップ接続だったので、LANカードは使っていなかった為、今回まで問題にならなかったのだろう。

私がいじってみても上手くカードを認識させられなかったので、使っていなかった他のカードに付け替えて無事リンクアップした。インターネットへの接続も問題なくできた。

　

父のこのパソコンだが、5年ほど前に私が使っていたものを渡したものなので非常にスペックが低い。確か、CPU が PentiumII233Mhz、メモリは SD-RAM64MB とかだったと思う。今回作業をしたときもとてつもなく重い。よくもまあこれほど重いものを使っていられるなあと感心してしまった。

私の手元に使っていなかった 128MB と 64MB のメモリがあったので、増設して 256MB とした。

メモリ増設の効果をこれほど実感できたことは無い。滅茶苦茶速くなった。

それでも十分遅いんだけど、それ以前のとてつもない状態から比べると速度向上に感涙してしまった。

父との間で、新しいパソコンに買い換えると言う話しも作業中に出ていたが、この速度向上を持って立ち消え。また、当分の間このスペックで使い続けることだろう。

■ 熱暴走

7月6日のところで書いた、パソコンの不調。原因がわかった。

原因は CPU の熱暴走。

クーラーをつけずに暑い部屋でパソコンを動かしていたところ、いつも以上にハングアップした。そこで、熱暴走を疑って、CPU やメモリのベンチマークソフトを走らせてみた。

結果は、CPU のテスト中にハングアップする、メモリ、ハードディスク、グラフィックボードのテストは問題なし。

BIOS で、CPU の温度を測ったところ、ケースを開けた状態で70度近くある。

冷却方法が悪いのかとも思ったが、クーラーをつけて、ケースを開け、CPU ファンが回っている状態で70度近くあるので、ハズレチップだったと考えるのが妥当だろう。

今回は新しく CPU を買うことにした。

もともと性能不足を感じての買い替えではないので、マザーボードなど CPU 以外はそのまま、現在のマザーボードで対応できる一番早い CPU にすることにした。これで選ぶと AthlonXP2600+(266MHz)になるが、これが16800円ほど、ひとつ下げて AthlonXP2400+ にすると8000円強。その為、AthlonXP2400+ にすることにした。現在通販で注文、商品が届くのを待っている。

これまで使っていた CPU は AthlonXP1900+ なので、単純に考えたら発熱量は大きくなる。これまで以上に悪い状態になったらどうしよう・・・。

2004-07-03 編集

■ ETC

ついにETCを購入した。

音声案内付きの中で一番安いものを購入。Panasonic の CY-ET800D。本体内蔵型でダッシュボードに取り付けるタイプ。音声案内以外は液晶表示も無くシンプルな製品。

イエローハットでクレジットガードごとETCカードを申し込むと5000円割引だったので、これで購入。

クレジットカードはセゾンのVISAカードを既に持っていたが、入会費も年会費も無料だったので作ってしまった。オリコのマスターカード。

ただこの直後にオートバックスからクレジットカードはもちろんETCカードの年会費も無料ってキャンペーンのダイレクトメールが届いた。

これまでETCカードの年会費が無料ってのは無かったので、それは仕方ないと思っていたんだけど・・。

ETCの年会費は525円とそれほど高くもないし、まあ良いか。

　

CY-ET800D は6月に買って申し込みもしていたので、つい先日クレジットカードとETCカードが届いた。

ちょうど今日高速道路を使って出かける用事があったので、使ってみた。
